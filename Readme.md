# AI Shop 프로젝트
다양한 AI 서비스를 활용한 shop 프로젝트입니다.

## ⚙ 기술 스택
- Spring boot 5.3.26
- Java 11
- Mybatis
- Mysql

## 📌 주요기능
### shop
- 회원가입, 로그인/로그아웃
- 상품 장바구니 담기, 삭제, 수량 조절
- 관리자: 상품등록, mall 등록, list 조회

### Kakao map API
- 서울, 부산, 제주도, 경주 맛집 정보 조회

### CFR
- 얼굴이미지의 유명인 닮은꼴 분석
- 얼굴이미지의 emotion, pose, gender, age 분석

### ChatBot
- 전체, 특정 사용자, 셀프 chat 구현

### OCR
- 사업자 등록증의 name, number 인식
- 카드의 유효기간, number 인식

### 날씨 조회
- 지역별 날씨 조회
